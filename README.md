## 주제 선정
Next.js의 실무적인 활용법을 익히기 위해, 추억의 게임 홈페이지를 직접 리뉴얼해 보았습니다.

## 작업 기한
- 2024.11.20 - 2024.12.20 (주요 기능 개발 완료) / 이후 유지보수 중

## 주요 기능
-  로그인, 회원가입, 커뮤니티(런너게시판), 커뮤니티(런너채팅방)
