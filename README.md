## 주제 선정
Next.js의 실무적인 활용법을 익히기 위해, 추억의 게임 홈페이지를 커스터마이징을 해보았습니다.

  
## 주요 기능
-  로그인, 회원가입, 커뮤니티(런너게시판, 런너채팅방)
